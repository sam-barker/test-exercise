# BT Test Exercise
The following repo is a solution to the Test Exercise for BT.

## Technologies
* Playwright
* TypeScript
* Prettier
* Husky

## How to run
To get up and running ensure you have
* NodeJS installed (LTS preferred)
* yarn installed globally

1. Clone the repo
2. Run `yarn install`
3. To run the tests, run `yarn test:headed`
